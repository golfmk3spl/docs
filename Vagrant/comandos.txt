Subindo vms definidas no arquivo Vagrant
$ vagrant up

Verificando Status das VMs definidas no arquivo Vagrant
$ vagrant status

Logando via SSH
$ vagrant ssh <nome-vm>

Destruindo VMs 
$ vagrant destroy <nome-vm>

Listando as imagens armazenadas para o Vagrant
$ vagrant box list

Removento uma imagem armazenada para o Vagrant
$ vagrant box remove <nome-imagem>
